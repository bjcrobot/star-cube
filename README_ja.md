# STAR CUBE

## イントロダクション

宇宙暦4227年 我々は新たに開発された観測技術 
R.T.P. System (Runic Tetra-Pixilation) により
物理学的定石を否定する奇妙な惑星の位置特定に成功した。

驚くべきことにそれは、立方体を基本構造とする様相を呈しているのである。
従来の宇宙物理学における「天体は重力により球体に収束する」という
常識を根底から覆す、まさにパラダイムシフトと言えよう。

これは、人類の宇宙科学における常識を覆す大発見であり、
観測に間違いがなければ、既存の全宇宙論文は、早急な書き換えを求められることとなるであろう。

さらに驚くべきはこの異形の天体は、無間にばらけながらも、
意志を持っているかのように自ら「正解」へと姿を整え始めていることだ。

そこで、我々はこの惑星群の調査プロジェクトを 「STAR CUBE」 と名付け個別の観測活動を開始することとした。
今後は、各関係機関との連携を続けながら引き続き調査を続ける。

- **Cosmic Horror**:

- **Fuel Depletion**:

## 概要

キューブパズルの実装検証のリポジトリです。
3部作の予定 マジックキューブを最初にリリースします。
ただ表示するだけでは、面白くないので宇宙空間の惑星に見立てています。
トップのポータル画面からキューブをクリックするとワープエフェクトによりマジックキューブ惑星に移動します。
Webのフロント技術のみで実装しているので、ローカルで起動可能、てPWAとしてインストール可能です。

## 技術スタック

- three.js
- JavaScript
- HTML/CSS

## 起動方法

1. Pythonが入っている環境で、`test_server.py` を実行します。
2. ブラウザで `http://localhost:8000` を開きます。

## PWA

- `manifest.webmanifest` と `sw.js` を同梱しています。
- HTTPS環境（またはlocalhost）でアクセスしてください。
- アイコン: `assets/images/icon-192.png`, `assets/images/favicon.png`
- スクリーンショット: `assets/images/pwa-screenshot.png`, `assets/images/pwa-screenshot-mobile.png`

## 操作方法

トップ画面
- クリック: キューブをワープ開始
- `M`: Musicのオン/オフ
- `A`: Aboutの開閉

magic-cube画面
- 右上の★: トップに戻る
- `M`: Musicのオン/オフ
- `S`: Soundのオン/オフ
- `R`: Reset
- `ArrowLeft/Right`: Speed調整
- `ArrowUp/Down`: HUDの開閉

## 補足

- マジックキューブの解法はKociembaアルゴリズムを使用しています。
- BGMは初回タップで解放されます。
